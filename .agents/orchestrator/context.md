# HeelsUp Storefront & API Enhancements Context

## Objective
Implement missing storefront features, backend reviews APIs, database & validation security, and performance optimizations.

## Targets
1. Storefront: Global Search, Product Filtering & Sorting, Reviews UI, Order Tracking Timeline.
2. Backend: Reviews POST/GET endpoints, validation & SQL sanitization.
3. Performance: preloading cache integrations (`frame_ant.js`).

## Existing codebase reference
- Vite + React in `frontend/`
- Cloudflare Workers in `src/`
- SQLite migrations in `migrations/`
